Implementando a pilha de JavaScript usando uma array

Resumo: este tutorial apresenta a estrutura de dados da pilha do JavaScript e mostra como usar um array como uma pilha.

Introdução à estrutura de dados da pilha

Uma pilha stack é uma estrutura de dados que contém uma lista de elementos. Uma pilha funciona com base no princípio LIFO, ou seja, Último a Entrar Last in, Primeiro a Sair First out, o que significa que o elemento adicionado mais recentemente é o primeiro a ser removido.

Uma pilha possui duas operações principais que ocorrem apenas no topo da pilha: empurrar push e estourar pop A operação push coloca um elemento no topo da pilha, enquanto a operação pop remove um elemento do topo da pilha.

A pilha de nomes vem da analogia com um conjunto de itens físicos, por exemplo, disco de DVD, livros, empilhados uns sobre os outros.

Uma pilha possui muitos aplicativos. Por exemplo, o mais simples é inverter uma palavra. Para fazer isso, você coloca uma palavra na pilha, letra por letra, e retira as letras da pilha.

As outras aplicações da pilha são o mecanismo de “desfazer” em editores de texto, análise sintática, chamada de função e conversão de expressão (infixo para pós-fixo, infixo para prefixo, pós-fixo para infixo e prefixo para infixo).

O tipo Array JavaScript fornece os métodos push() e pop() que permitem que você use uma matriz(array) como uma pilha.
